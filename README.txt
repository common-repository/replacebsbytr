Replace Blogsearch by Technorati incoming links on the Dashboard

== Description ==

* English
This plugin replace the Blogsearch incoming links in the Dashboard by Technorati results, like in WordPress <2.3

* Français
Ce plugin remplace les liens entrants du tableau de bord en provenance de Blogsearch par les résultats de Technorati, à la façon de WordPress dans les version inférieurs à 2.3

== Installation ==

* English
1. Upload `ReplaceBSbyTR` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Nothing else

* Français
1. Placer le dossier 'ReplaceBSbyTR' dans le dossier '/wp-content/plugins/' de votre blog
2. Activer le plugin depuis le menu 'Plugins' de votre console WordPress
3. Rien de plus
